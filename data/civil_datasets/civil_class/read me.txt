---------------
DATA FORMAT: 
---------------
-	city
[cell] each cell contains the name of the corresponding task (i.e. city)

-	labels
[cell] each cell contains the name of the corresponding event subtype (i.e. primary population type)

-	latitude
[double] the latitude of the corresponding task

-	longitude
[double] the longitude of the corresponding task

-	X
[double] the input data, a 3-D martix where each dimension represents [task, sample, feature vector]

-	Y
[double] the target class labels, a 2-D martix where each dimension represents [task, sample]
